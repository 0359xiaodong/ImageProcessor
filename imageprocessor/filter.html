Filter

Applies a filter to the current image. Use the MatrixFilters class to assign the correct filter

Filter(IMatrixFilter matrixFilter)

Parameters

matrixFilter
The IMatrixFilter of the filter to add to the image.
Filter(string filterName)

Parameters

filterName
The name of the filter to add to the image.

Please note that the Filter(string filterName) overload is now obsolete and will be removed in the next major version.

Available filters are:

  • blackwhite
  • comic
  • lomograph
  • greyscale
  • polaroid
  • sepia
  • gotham
  • hisatch
  • losatch
  • invert
