Filter

Applies a filter to the current image.

Filter(string filterName)

Parameters

filterName
The name of the filter to add to the image.

Available filters are:

  • blackwhite
  • comic
  • lomograph
  • greyscale
  • polaroid
  • sepia
  • gotham
  • hisatch
  • losatch
  • invert
