Save

Saves the current image to the specified file path or MemoryStream.

Save(MemoryStream memoryStream)

Parameters

memoryStream
The System.IO.MemoryStream to save the image information to.
Save(string filePath)

Parameters

filePath
The absolute path to save the image to.
