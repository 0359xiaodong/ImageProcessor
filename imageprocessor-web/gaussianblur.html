GaussianBlur

Uses a Gaussian kernel to blur the current image. Simply pass the desired kenel size to the processor.

Additional optional properties are also available
  • sigma (defaults to 1.4)
  • threshold (defaults to 0)

All gaussian blur requests have limiters to the given values set in the processing.config file to help prevent DoS attacks.

Requests are passed thus: 

http://your-image?blur=11,sigma-1.5,threshold-10
