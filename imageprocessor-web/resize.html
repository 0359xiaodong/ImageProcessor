Resize

Resizes the current image to the given dimensions.

ImageProcessor.Web allows you to scale images both up and down with an excellent quality to size ratio. A maximum width and height can be set in the configuration to help protect you from DoS attacks.

Four different types of resizing methods are available:

  • Pad (default)
  • Crop
  • Max
  • Stretch

Requests are passed thus:

Resize Pad

Passing a single dimension will automatically preserve the aspect ratio of the original image. If the requested aspect ratio is different then the image will be padded to fit. You can also optionally fill the background color for image types without transparency.

http://your-image?width=600
http://your-image?height=250
http://your-image?width=600&height=250
http://your-image?width=600&height=250&bgcolor=fff

Resize Crop

Crop mode also offers optional anchor positions:

  • Center (default)
  • Top
  • Right
  • Bottom
  • Left

http://your-image?width=600&height=250&mode=crop
http://your-image?width=600&height=250&mode=crop&anchor=top

Resize Max

The maximum possible height or width is used that allows the correct aspect ratio.

http://your-image?width=600&height=250&mode=max

Resize Stretch

The image is stretched to wit the given dimensions..

http://your-image?width=600&height=250&mode=stretch
