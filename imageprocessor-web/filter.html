Filter

Applies Instagram style filters to the current image.

Available filters starting from top-left and going across each row are:

  • blackwhite
  • comic
  • lomograph
  • greyscale
  • polaroid
  • sepia
  • gotham
  • hisatch
  • losatch
  • invert

Requests are passed thus: 

http://your-image?filter=comic
