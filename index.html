Imageprocessor

A .NET library for on-the-fly processing of image files.

What is it?

Imageprocessor is a lightweight library written in C# that allows you to manipulate images on-the-fly using ASP.NET 4.0.

It's lighting fast, extensible, easy to use, comes bundled with some great features and is fully open source.

DOWNLOAD ON NUGET

Installation

Installation is simple. A Nuget package is available here which makes it possible to install and configure straight from Visual Studio.

Alternatively, download the zip file from here and copy the two binaries into your bin folder. Then copy the example configuration values from the web.config in the testing website from within the solution into your web.config to enable the processor.

Features

Imageprocessor has lots of cool features that make it a great choice for developers like intelligent caching, a fluent, easy to use API and much more.

All these features can be combined and called in any order due to regex based parsing and have an easy to remember syntax. You can even use it on remote images just add the prefix /remote.axd? to your image request.

Here's a few examples:

  • caching
  • resizing
  • cropping
  • alpha transparency
  • filter effects
  • brightness
  • contrast
  • saturation
  • watermark
  • rotate
  • format
  • vignette
  • quality
  • flip
  • rounded corners

Caching

Imageprocessor caches any processed image both in the browser and on the server for a length of your choosing. The server cache intelligently stores up to 12,960,000 images and silently updates itself should the original image change or the cache expires.

This saves massive amounts of processing power and bandwidth that help to keep your website snappy.

Resizing

Imageprocessor can resize bmps, jpegs, pngs and gifs (gifs require full trust) up or down producing the highest quality output at the lowest possible filesize.

Images passed only one parameter will automatically maintain their aspect ratio.

The library also protects you from denial of service attacks by ignoring requests to generate images larger than a maximum value you set.

Simply pass the desired pixel width or height (without the px).

Requests are passed thus: 

http://your-image?width=250
http://your-image?height=180

Cropping

Imageprocessor can crop a given image and serve only the part you want. Simply pass the top-left and bottom-right coordinates and the processor will work out the rest.

Requests are passed thus: 

http://your-image?crop=x-y-x2-y2

Alpha

Imageprocessor can adjust the alpha transparency of png images. Simply pass the desired percentage value (without the '%') to the processor.

Requests are passed thus: 

http://your-image?alpha=75

Filters

Everybody loves adding filter effects to photographs so we've baked some popular ones into Imageprocessor.

Filters starting from top-left and going across each row are:

  • blackwhite
  • comic
  • lomograph
  • greyscale
  • polaroid
  • sepia
  • gotham
  • hisatch
  • losatch
  • invert

Requests are passed thus: 

http://your-image?filter=comic

Brightness

Imageprocessor can adjust the brightness of images. Simply pass the desired percentage value (without the '%') to the processor.

Requests are passed thus: 


// Increasing
http://your-image?brightness=20


// Decreasing
http://your-image?brightness=-20

// Increasing

Saturation

// Increasing

Watermark

Imageprocessor can add watermark text to your images with a wide range of options:

  • text
  • size (px)
  • font (any installed font)
  • opacity (1-100)
  • style (bold|italic|regular|strikeout|underline)
  • shadow
  • position (x,y)

Requests are passed thus: Each parameter is optional. 

http://your-image?watermark=text-testing|color-fff|size-36|style-italic|opacity-80|position-30-150|shadow-true|font-arial

Rotate

Imageprocessor can rotate your images without clipping. You can also optionally fill the background color for image types without transparency.

Requests are passed thus: 

http://your-image?rotate=26

Color syntax: 

http://your-image?rotate=angle-54|bgcolor-fff

Format

Imageprocessor allows you to change the format of image on-the-fly.

Supported file formats are:

  • jpg
  • bmp
  • png
  • png8
  • gif
  • tif

Requests are passed thus: 

http://your-image?format=png8

Vignette

Imageprocessor can also add a vignette effect to images.

Requests are passed thus: 

http://your-image?vignette=true

Quality

Whilst Imageprocessor delivers an excellent quality/filesize ratio it also allows you to change the quality of jpegs on-the-fly.

Requests are passed thus: 

http://your-image?quality=65

Flip

Imageprocessor can flip you images both horizontally and vertically.

Requests are passed thus: 

http://your-image?flip=horizontal
http://your-image?flip=vertical

Rounded Corners

Imageprocessor can round the corners of your images. You can also optionally fill the background color for image types without transparency and choose which corners to round.

Requests are passed thus: 

http://your-image?roundedcorners=26

Color syntax: 

http://your-image?roundedcorners=radius-26|bgcolor-fff

Independent Corners syntax: 

http://your-image?roundedcorners=radius-26|tl-true|tr-false|bl-true|br-false
