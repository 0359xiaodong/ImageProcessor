Imageprocessor

A library for on-the-fly processing of image files with Asp.NET

Download this project as a .zip file Download this project as a tar.gz file

Imageprocessor is a lightweight library written in C# that allows you to manipulate images on-the-fly using Asp.NET 4.0.

It comes in two flavours:

  • ImageProcessor.dll which contains the raw fluent api for manipulating images
  • ImageProcessor.Web.dll which contains a HttpModule for intercepting image requests and processes and caches them based on the given querystring parameters.

Resizing

Resizing an image is easy. Simply add the parameter ?width=yourvalue to the end of the image path and the image will be served at the given width. Unless instructed otherwise with a height=yourvalue parameter this image will automatically maintain the aspect ratio of the original image.

As you can see the images retain excellent sharpness whilst maintaining a low file size. This was the product of a lot of experimentation with the System.Drawing.Graphics object.

Filters

The processing library contains a few filters to change the appearance of images.

BlackWhite filter=blackwhite

Greyscale filter=greyscale

Comic filter=comic

Sepia filter=sepia

The next two filters copy popular camera effects. An additional parameter has been added to produce vignetting.

Polaroid filter=polaroid&vignette=true

Lomograph filter=lomograph&vignette=true

Installing

Installation is simple. Download the zip file from here and copy the two binaries into your bin folder. Then copy the example configuration values from the config.txt into your web.config to enable the processor. A Nuget package will be created once I've read the manual to allow simpler installation in the future.
