What is it?

Imageprocessor is a lightweight library written in C# that allows you to manipulate images on-the-fly using Asp.NET 4.0.

It's fast, extensible, easy to use, comes bundled with some great features and is fully open source.

Installation

Installation is simple. Download the zip file from here and copy the two binaries into your bin folder. Then copy the example configuration values from the config.txt into your web.config to enable the processor. A Nuget package will be created once I've read the manual to allow simpler installation in the future.

Features

Imageprocessor has lots of cool features that make it a great choice for developers like intelligent caching, a fluent, easy to use api and much more.

All these features can be combined and called in any order due to regex based parsing and have an easy to remember syntax.

Here's a few examples:

  • caching
  • resizing
  • cropping
  • alpha transparency
  • filter effects

Caching

Imageprocessor caches any processed image both in the browser and on the server for a length of your choosing. The server cache intelligently stores up to 8000 images and silently updates itself should the original image change or the cache expires.

This saves massive amounts of processing power and bandwidth that help to keep your snappy.

Resizing

Imageprocessor can resize bmps, jpegs, pngs and gifs (gifs require full trust) up or down producing the highest quality output at the lowest possible filesize.

Images passed only a width parameter will automatically maintain their aspect ratio.

The library also protects you from denial of service attacks by ignoring requests to generate images larger than a maximum value you set.

Requests are passed thus:
http://yourimage?width=yourwidth

Cropping

Imageprocessor can crop a given image and serve only the part you want. Simply pass the top-left and bottom-right coordinates and the processor will work out the rest.

Requests are passed thus:
http://yourimage?crop=x-y-x2-y2

Alpha

Imageprocessor can adjust the alpha transparency of png images. Simply pass the desired percentage value (without the '%') to the processor.

Requests are passed thus:
http://yourimage?alpha=percentage

Filters

Everybody loves adding filter effects to photgraphs so we've baked some popular ones into Imageprocessor.

Filters starting from top-left and going across each row are:

  • blackwhite
  • comic
  • lomograph
  • greyscale
  • polaroid
  • sepia
  • gotham
  • hisatch
  • losatch

Requests are passed thus:
http://yourimage?filter=name
